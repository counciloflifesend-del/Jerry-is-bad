# Jerry-is-bad

A playful single-page website that says "Jerry is bad" in many random typographic ways.

## Open locally

Open `index.html` in a browser, or serve the folder with a small local web server:

```sh
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Controls

- Click anywhere for a random option.
- Press `Space` or `Enter` for another random option.
- Use the glass button in the corner to enter or exit fullscreen.
